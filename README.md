# TI-83 Style Finance Calculator

A simple mobile-friendly finance calculator inspired by classic TI-83 finance workflows.

## Features
- **TVM Solver** (solve for N, I/Y, PV, PMT, FV)
- **Loan Payment calculator**
- **Amortization schedule generator**
- Responsive design for phone testing

## Run locally
Just open `index.html` in a browser.

## Deploy with GitHub Pages
1. Push this repo to GitHub.
2. In GitHub: **Settings → Pages**
3. Source: **Deploy from branch**
4. Branch: `main` / folder: `/ (root)`
5. Wait for deployment and open the Pages URL.
